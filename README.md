# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas + JavaScript 实现的经典坦克大战游戏，可直接在浏览器中运行。

## 🎯 游戏特色

- 🕹️ **经典玩法** — 控制坦克消灭所有敌方坦克即可过关
- 🗺️ **5 个关卡** — 内置 5 张精心设计的地图，通关后自动生成新地图
- 💥 **三种砖块** — 砖墙（可击毁）、钢墙（不可击毁）、水域（不可通过）
- 🎁 **四种道具** — 加速⚡、护盾🛡、三连发💥、加命❤
- 🎨 **视觉效果** — 粒子爆炸、屏幕震动、护盾光环、弹道尾迹
- 🔊 **音效反馈** — 使用 Web Audio API 生成射击、爆炸等音效
- 📱 **触屏支持** — 支持手机和平板触屏操作
- ⏸️ **暂停功能** — 按 P 键随时暂停/继续

## 🎮 操作方法

| 按键 | 功能 |
|------|------|
| `WASD` / `方向键` | 移动坦克 |
| `空格` / `J` | 射击 |
| `P` | 暂停/继续 |
| `R` | 重新开始（游戏结束时） |

### 移动端

- 触摸屏幕边缘区域控制移动方向
- 触摸中心区域进行射击

## 📝 游戏规则

- 消灭关卡中所有敌方坦克即可进入下一关
- 每关敌人数量和速度会逐渐增加
- 玩家有 3 条命，被击中会扣一条命并短暂无敌
- 击毁敌人有概率掉落道具
- 分数 = 100 × 当前关卡数 × 击杀数 + 500 × 通关奖励
